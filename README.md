# -Electric-Vehicle-Recharge-Bunk
This project is a full-stack web application developed using the MERN Stack (MongoDB, Express.js, React.js, and Node.js) that helps users locate, view, and manage electric vehicle (EV) recharge stations. The application aims to support the growing EV ecosystem by providing a digital platform for discovering nearby charging bunk locations and their details.

The system includes a user-friendly frontend, a secure backend API, and a database-driven architecture to store recharge bunk information. It ensures smooth data flow between client and server while maintaining scalability and performance.
